# PLO Trainer Matrix 핸드 레인지 선택 UI - 상세 분석

> 영상 구간: 32분 10초 ~ 33분 10초  
> 출처: "Exploring PLO Trainer's New Design" by JNandez  
> 분석일: 2026년 2월 12일

---

## 1. Matrix 뷰 개요

PLO Trainer의 Matrix는 Study 모드 우측 패널에서 **Range 탭 옆에 위치한 Matrix 탭**을 클릭하면 활성화됩니다. PLO의 4장 핸드를 **13×13 그리드 매트릭스**로 압축하여 직관적으로 레인지를 탐색할 수 있게 해주는 핵심 기능입니다.

일반적인 텍사스 홀덤의 2장 핸드와 달리 PLO는 4장의 카드를 처리해야 하므로, **2단계 선택 방식**으로 동작합니다.

---

## 2. 2단계 선택 메커니즘 (PLO 특화)

### 2.1 1단계: 첫 번째 2장 선택 (Highest Two Cards)

최초 Matrix 화면은 표준 13×13 그리드로 표시됩니다. 각 셀은 PLO 4장 중 **가장 높은 2장의 조합 (Highest two cards)**을 나타냅니다.

- **행/열 레이블:** A, K, Q, J, T, 9, 8, 7, 6, 5, 4, 3, 2 (총 13개)
- **대각선 셀:** AA, KK, QQ, JJ, TT, 99, 88 등 페어 핸드 (특별 구분색 표시)
- **상단 우측 셀 (e.g., AK, AQ):** Good Suit 조합 (수티드됨)
- **하단 좌측 셀 (e.g., KA, QA):** Bad Suit 조합 (오프수트)
- **안내 텍스트:** "Select your pair or highest two cards. Top Right: Good Suit / Bottom Left: Bad Suit"

#### 셀 컬러 코딩 (1단계)

각 셀은 해당 핸드 그룹의 솔버 액션 비율에 따라 컬러가 변합니다:

| 색상 | 의미 | 설명 |
|---|---|---|
| 초록/청록계 (#4CAF50) | Fold 빈도 높음 | 솔버가 대부분 Fold를 추천하는 핸드 그룹 |
| 파란/남색계 (#42A5F5) | Call/Check 빈도 높음 | 패시브 액션이 주를 이루는 핸드 그룹 |
| 분홍~빨간계 (#E57373) | Bet/Raise 빈도 높음 | 액티브 액션이 주를 이루는 핸드 그룹 |
| 혼합색 (새턴) | 믹스드 전략 | 빈도가 믹스된 경우 색이 혼합되어 표시 |

### 2.2 2단계: 나머지 2장 선택 (Other Two Cards)

1단계에서 셀을 클릭하면(e.g., "A8" 셀), Matrix가 **2단계 선택 화면으로 전환**됩니다. 이제 나머지 2장을 선택하는 그리드가 표시됩니다.

- **안내 텍스트 변경:** "Select your other two cards. Top Right: Good Suit / Bottom Left: Bad Suit"
- **"Back" 버튼:** 좌측 상단에 나타나며, 1단계로 돌아갈 수 있음
- **선택된 1단계 카드 표시:** 좌측 상단에 선택된 카드가 칩 형태로 표시됨 (e.g., [A][8])
- **그리드의 색상 변화:** 1단계와 동일한 컬러 코딩이지만, 선택한 첫 2장과 조합되는 나머지 2장의 전략 빈도로 색상이 바뀜
- **"Switch to any cards" 버튼:** 우측 상단에 위치. 2단계 그리드에서 제한 없이 임의의 카드 조합을 선택할 수 있는 모드로 전환
- **결과:** 2단계 셀 클릭 시 → 해당 4장 조합의 핸드들이 좌하단 Hand 리스트에 필터링되어 표시

## 3. Diagonal Suit Logic Separation (수트 필터 시스템)

Matrix 우측에 **"Diagonal suit logic separation"** 옵션이 있으며, PLO 특성상 수트 조합이 핸드 가치에 큰 영향을 미치므로 중요한 필터입니다:

| Suit 필터 옵션 | 설명 |
|---|---|
| **Single suited** | 4장 중 2장만 같은 수트 (e.g., A♠ K♥ 8♠ 5♣) |
| **Double suited** (기본값) | 2쌍이 각각 같은 수트 (e.g., A♠ K♥ 8♠ 5♥) - 가장 가치 높은 수트 구성 |
| **Triple suited** | 3장이 같은 수트 (e.g., A♠ K♠ 8♠ 5♥) |
| **Monotone / Rainbow** | 4장 모두 같은 수트 / 모두 다른 수트 |

**핵심 포인트:** Suit 필터를 변경하면 Matrix 전체의 색상과 핸드 리스트가 즉시 업데이트됩니다. 영상에서 JNandez가 기본값으로 "Double suited"를 선택하여 사용했습니다.

---

## 4. Normal / Advanced 모드 전환

Matrix 우측 상단에 **Normal | Advanced** 토글 버튼이 있습니다:

| 모드 | 설명 |
|---|---|
| **Normal** | 기본 13×13 그리드로 2단계 선택 방식. 일반 사용자에게 적합한 간결한 인터페이스 |
| **Advanced** | 더 세분화된 핸드 선택이 가능한 고급 모드. 특정 수트 콤보나 연결성 등 더 깊은 필터링 |

---

## 5. 좌하단 Hand 리스트 연동

Matrix에서 셀을 선택하면 좌하단의 **Hand 리스트가 실시간으로 필터링**되어 해당 조합의 구체적인 핸드들이 표시됩니다.

### 5.1 Hand 리스트 컬럼 구성

| 컬럼 | 예시 | 설명 |
|---|---|---|
| **Hand** | A♠ 8♥ 7♠ 5♥ | 4장의 카드가 수트별 컬러로 표시 |
| **Frequency** | 100% / 96% / 37% | 솔버가 해당 핸드를 플레이하는 빈도. 초록/빨간 바 그래프로 시각화 (Fold=빨간, Call/Bet=초록) |
| **maxEV** | 0.79 / 0.66 / 0.42 | 해당 핸드의 최대 기대값 (EV). 높을수록 수익성 높은 핸드 |

### 5.2 Categories 연동

Hand 리스트 위에는 **Categories 바 그래프**가 함께 표시되어 선택한 핸드 그룹의 전략 분포를 볼 수 있습니다:

- **탭 전환 버튼:** "Highest two cards" / "Double suited" / "Good suits" 등으로 핸드 리스트의 정렬/필터 기준 변경
- **액션 비율 바:** 초록(Fold) / 빨간(Bet) 비율이 수평 바로 표시되어 전체 레인지의 전략 경향을 한눈에 파악
- **영상 예시:** A8 + 75 조합 선택 시 → 좌하단에 A♠8♥7♠5♥, A♠8♥7♠5♠ 등 2개 핸드만 표시됨. Frequency 100%, maxEV 0.70/0.66

## 6. 전체 UX 플로우 정리

**Matrix를 통한 핸드 레인지 탐색 플로우:**

1. **Study 모드 진입 →** 시나리오 선택 (Preflop/Flop/Turn/River 노드)
2. **Matrix 탭 클릭 →** 13×13 그리드 표시 (1단계: Highest two cards)
3. **Suit 필터 선택 →** Single/Double/Triple suited 또는 Rainbow 선택
4. **1단계 셀 클릭 (e.g., A8) →** 2단계 그리드로 전환 (나머지 2장 선택)
5. **2단계 셀 클릭 (e.g., 75) →** 좌하단 Hand 리스트에 A8+75 조합 핸드들 필터링 표시
6. **개별 핸드 분석 →** Frequency, maxEV 확인 + Range 테이블과 교차 참조
7. **Train 버튼 클릭 →** 해당 조건으로 바로 훈련 세션 시작 가능
8. **Back 버튼 / 탭 전환 →** 언제든 이전 단계로 복귀 가능

## 7. 우리 서비스 개발 기획 포인트

### 7.1 필수 구현 요소

1. **13×13 Matrix 그리드 UI:** 각 셀이 액션 빈도에 따라 컬러가 동적으로 변하는 그리드. 호버 시 툴팁으로 액션 비율 상세 표시
2. **2단계 선택 로직:** PLO 4장 핸드를 2+2 방식으로 선택. 첫 2장 → 나머지 2장 순차적 선택
3. **Suit 필터 시스템:** Single/Double/Triple suited, Monotone/Rainbow 필터를 토글식으로 제공
4. **Hand 리스트 실시간 연동:** Matrix 셀 선택 시 좌하단 리스트가 즉시 필터링. Frequency 바 + maxEV 컬럼 포함
5. **Train 버튼 연동:** Matrix 우측 상단의 "▶ Train" 버튼으로 현재 보고 있는 스팟의 훈련 세션 바로 시작

### 7.2 차별화 포인트

- **드래그 선택 모드:** PLO Trainer는 클릭만 지원. 우리는 드래그로 범위 선택(e.g., A8~A5 범위를 한 번에 선택) 기능을 추가하면 UX 개선
- **핸드 비교 모드:** 2개 이상의 셀을 선택하여 핸드 그룹간 EV/Frequency 비교 기능
- **히트맵 모드:** Matrix 컬러를 단순 카테고리색 대신 연속적인 그라디언트 히트맵으로 표시하여 EV 가치를 더 직관적으로 시각화
- **5-Card PLO 확장:** 5카드 PLO의 경우 3단계 선택(2+2+1) 또는 다른 그룹핑 로직 필요. 확장성 고려 설계

### 7.3 기술적 고려사항

- **성능:** 13×13 = 169개 셀이 동시에 액션 빈도 기반으로 컬러를 계산해야 함. CSS 변수나 Canvas/WebGL로 렌더링 최적화 필요
- **반응성:** Suit 필터 변경/셀 클릭 시 100ms 이내 응답 필요. 솔루션 데이터 프리로딩 또는 캐싱 전략 필요
- **모바일 대응:** PLO Trainer는 모바일 호환. 13×13 그리드를 작은 화면에서도 터치 가능하게 설계 (핀치 줌, 스크롤 등)
- **데이터 구조:** Matrix 데이터는 `[rank1][rank2][suit_type]` → `{action_frequencies, ev, combos}` 형태로 저장. 2단계는 `[rank1][rank2][rank3][rank4][suit_type]`으로 확장
